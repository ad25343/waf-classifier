Work Alignment Framework

The WAF framework categorizes all work into 8 categories across two types: Run (keeping things going) and Change (building new things). Each category has a color that indicates investment type and priority level.

Gray — KTLO
Black — Maintenance
Red — Regulatory
Orange — Enterprise
Yellow — Divisional
Green — Other
